Pathfinding Visualizer

This web application allows a user to visualize industry-standard pathfinding algorithms in a highly intuitive manner. After animating a search, the user can adjust the start and target nodes to automatically recompute the search-space and shortest-path. The graph can also be populated with randomly-generated patterns.
Technologies: JavaScript, HTML5, CSS.

Facial Expression Classifier

This is a deep-learning program developed with python, tensorflow, and keras, used for the detection of human facial expressions. Expressions are categorized into one of five categories with up to 91.7% accuracy (tested on over 10,000 images).
Technologies: Python, TensorFlow, Keras, OpenCV, Pygame.
